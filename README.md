DRG Reflex is a fully **offline**, local-first system: a Meta Quest VR app plus a desktop
companion (Windows &amp; macOS) that transfers driving-hazard videos to the headset and
reads back precise reaction times — no server, no cloud, no recurring cost.

## ⬇️ Download

| Platform | Download |
|----------|----------|
| **Windows** | [Installer (.exe)](../../releases/latest) |
| **macOS** | [Installer (.dmg)](../../releases/latest) |
| **Meta Quest** (VR app) | [APK](../../releases/tag/vr-apk) |

> 💡 The desktop app installs the VR app onto a USB-connected Quest **for you** and keeps it
> up to date automatically. The standalone APK above is only for manual sideloading.

## Install

- **Windows** — run the installer. The app checks for updates on launch and updates itself.
- **macOS** — open the `.dmg` and drag **DRG Reflex** into *Applications*.
- **Meta Quest** — connect the headset by USB and use the desktop app's **Install on Quest**,
  or sideload the APK manually.

## Activation

DRG Reflex is activated with an offline, cryptographically signed license key — no internet
required. Contact your DRG Reflex provider for a key.
